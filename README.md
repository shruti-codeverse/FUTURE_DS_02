# FUTURE_DS_02
📊 Telco Customer Churn Analysis Dashboard

📌 Project Overview:
Customer churn is a major challenge for telecom companies.
This project analyzes customer data to identify patterns and factors contributing to churn using Power BI.

The goal of this dashboard is to:
  Clean and organize customer or subscription data
  
 Analyze:
      . Churn rates and retention trends
      . Customer cohorts (by signup month, plan, or region)
       . Customer lifetime patterns
       . Identify key retention drivers and churn reasons

📁 Dataset Description:

  Total Customers: 7,043
  Total Churned Customers: 1,869
  Churn Rate: 26.54%
  Average Tenure: 32.37 months

  🔍 Key Insights:
  
  Customers with Month-to-Month contracts show higher churn.
  Electronic Check payment method has higher churn compared to automatic payments.
  Customers with low tenure are more likely to churn.
  Fiber optic users show notable churn patterns.

  🛠 Tools & Technologies Used:

  Power BI
  DAX (for calculated measures)
  Data Cleaning & Transformation
  Data Visualization
  Business Analysis
